"Autenticação"
- Login salvo em localStorage: nio_token='logado', nio_usuario=nome, nio_login_ts=timestamp
- Sessão válida por 8 horas. Se expirada, redireciona para login.html

"Observações importantes"
- GAS usa doGet() para leituras e doPost() para gravações
- Posts usam mode:'no-cors' — não é possível ler a resposta do POST diretamente
- Sempre adicionar &t=Date.now() nas chamadas GET para evitar cache
- Ao alterar o GAS: Salvar → Implantar → Gerenciar implantações → Editar → Nova versão → Implantar
- A URL do GAS só muda se criar um NOVO deploy — nesse caso atualizar todos os HTMLs e este arquivo
